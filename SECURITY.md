# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it privately by email to Trystan Lea at trystanlea@openenergymonitor.org.

Please include as much detail as possible so the issue can be reproduced and assessed quickly, such as:

- a description of the problem
- steps to reproduce it
- the affected version or module
- any potential impact

Please do not disclose the issue publicly, including in GitHub issues, until it has been reviewed and a fix or mitigation is available.

Reports will be reviewed as soon as possible. Thank you for helping keep Emoncms and its users secure.
